# Project 6 - Customer Churn Analysis for Telecom

## Tools Used:
- Python (Pandas, Sklearn, Seaborn)
- SQLite (SQL queries)
- Google Colab

## Dataset: Telco Customer Churn (7032 customers)

## Deliverables:
- Python ML Notebook
- Customer Churn Report (PDF)
- PowerPoint Presentation (10 slides)

## Key Findings:
- Month-to-month contracts highest churn
- High monthly charges = high churn risk
- ROC-AUC Score: 0.84
- Churn reduction potential: 5-10%
